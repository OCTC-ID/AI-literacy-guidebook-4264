Using the playbook

Start with one useful connection

For faculty

Select the BE habit that most naturally fits an existing assignment, discussion, or course decision. The guides are coaching resources, not required rubrics.

For course design

Use the developmental progression to decide what students need now, then choose one quick win or reflection prompt that makes the habit visible.
